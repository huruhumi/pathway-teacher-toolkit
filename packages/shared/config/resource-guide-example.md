## 二、单元内部结构：Trail 1 + Trail 2

每个单元围绕一个 **Big Question**，分两条学习路径探究：

```
Unit Big Question
├── Trail 1（子主题 A）
│   ├── Vocabulary + Phonics
│   ├── Reading Strategy + Nonfiction Reading
│   ├── Comprehension
│   ├── Grammar Point 1
│   ├── Listening
│   ├── 🔵 Speaking（Trail 1 专属）
│   └── Return to Big Question
│
└── Trail 2（子主题 B）
    ├── Vocabulary + Phonics
    ├── Reading Strategy + Fiction Reading
    ├── Comprehension
    ├── Grammar Point 2
    ├── Listening
    ├── 🟠 Project（Trail 2 专属）
    ├── 🟠 Unit Test（Trail 2 专属）
    └── Return to Big Question
```

> **设计意图**：Trail 1 用非虚构文本建立事实基础 + 口语输出；Trail 2 用虚构文本激发想象 + 项目实践。两条路径提供不同视角回答同一个大问题。

## 三、课时安排方案

### 方案 A：1×90min/周（20 周完成 Starter）

每单元 2 节课（180min），每 2 单元后 1 节复习课。

#### Session 1 — Trail 1（90min）

| 阶段 | 时长 | 目标 | 活动设计 |
|------|-----|------|---------|
| Warm-up | 10min | 激活旧知，引入 Big Question | Big Question 讨论 + 图片预测 |
| Vocabulary | 15min | 掌握 Trail 1 核心词汇 | TPR全身反应 + 实物/图片闪卡 + 词汇chant |
| Phonics 1 | 10min | 识别目标字母/音素 | 字母歌 + 空中书写 + Phonics Hopscotch |
| Reading 1 | 20min | 理解非虚构文本大意 | 教师带读 + 图片走查 + 理解提问（Yes/No） |
| Grammar 1 | 15min | 初步感知语法结构 | Chant drill + 教师示范 + 全班齐练 |
| Listening | 10min | 捕捉关键信息 | 播放音频 + 指图/举手互动 |
| Speaking | 10min | 运用口语策略进行简单交际 | Pair work: 用本课句型互问互答 |

#### Session 2 — Trail 2（90min）

| 阶段 | 时长 | 目标 | 活动设计 |
|------|-----|------|---------|
| Review Warm-up | 10min | 巩固 Trail 1 词汇和语法 | 闪卡竞赛 + LC互动任务回顾 |
| Vocabulary | 10min | 扩展 Trail 2 词汇 | 图片配对 + 词汇墙更新 |
| Phonics 2 | 10min | 巩固新一组字母/音素 | 发音模仿 + 拼读游戏 |
| Reading 2 | 15min | 理解虚构文本并建立联系 | 角色朗读 + 故事排序 + 预测结局 |
| Grammar 2 | 15min | 运用第二个语法点 | 句型替换练习 + 信息差活动 |
| Listening | 10min | 听取详细信息 | 音频播放 + 完成任务卡 |
| Project | 15min | 综合运用单元语言创作作品 | 项目启动+课堂制作+简单展示 |
| Wrap-up + Assessment | 5min | 检查单元核心目标达成 | Mini-check: 3个核心问题快速应答 |

| 阶段 | 时长 | 目标 | 活动设计 |
|------|-----|------|---------|
| Song & Visual | 20min | 在情境中复习两单元语言 | 教材大插图活动 + 单元歌曲 |
| Vocab Game | 20min | 串联两单元核心词汇 | 词汇接龙/Bingo/Simon Says |

| Formative Assessment | 15min | 诊断薄弱环节 | 自评卡 + 教师观察记录 |

### 方案 C：智能判断（自定义课时）

若前端选择的课时不符合方案 A/B，AI 按以下规则自动分配：

1. **计算总可用分钟数** = 周数 × 每周课时分钟
2. **每单元分配** = 总分钟 ÷（单元数 + 复习课数）
3. **分配策略**：
   - 每单元 ≤ 120min → 压缩为 1 节课，Trail 1+2 合并，项目推到 LC
   - 每单元 120-200min → 方案 A 模式（2 节课）
   - 每单元 200-300min → 方案 B 模式（4 节课）
   - 每单元 > 300min → 扩展版，增加深度阅读/写作环节
4. **复习课** = 总课次的 15-20% 分配给复习

## 七、AI 生成调用策略

### 生成大纲（Curriculum）时

1. 读取本指南 → 获取单元数、Trail 结构、Scope & Sequence
2. 根据前端选择的课时方案（A/B/C）确定拆分方式
3. 查询 **Lesson Planner** → 对齐每课教学内容
4. 查询 **Pacing Guide** → 参考官方节奏建议
5. **严禁跨单元合并内容**，每课必须标注 Trail 1 或 Trail 2

### 生成 Lesson Plan 时

1. 读取本指南 → 定位该课所在 Unit + Trail
2. 查询 **Student's Book** → 提取对应阅读文本和练习
3. 查询 **Lesson Planner** → 提取教学步骤和活动建议
4. 每个阶段（stage）必须包含：**阶段目标** + **实操活动** + **教学提示**
5. 按需查询 **Audio Scripts**（听力）/ **Rubrics**（评估）

### 生成 Assignment 时

1. 查询 **Workbook** → 提取对应单元的书面练习
2. 自动生成作业内容（填空、描摹、配对等）

### 生成 LC 互动任务时

1. 只生成互动型任务（游戏/录音/挑战）
2. 参考当课词汇和语法生成对应互动内容
